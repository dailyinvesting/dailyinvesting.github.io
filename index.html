打工人的0基础 Codex 手册：6 个实战场景（附可复制 Prompt）

一个打工人写给打工人的实操指南。不是 AI 概念，不是趋势分析， 而是你今天就能复制使用的真实工作流。

1 小时工作，7 小时摸鱼。
前提是：把重复劳动交给 AI。

写在前面：这篇文章能给你什么

2025 年，OpenAI 官方发布了一篇文章：《Codex 在工作中的十大使用场景》。 这件事本身就是一个信号：

一个原本被定义为“程序员 AI 工具”的产品， 开始正式面向普通职场人。

重点甚至已经不是“写代码”，而是：

  • 整理消息
  • 管理任务
  • 自动生成周报
  • 分析 Excel
  • 监控竞品
  • 自动跑后台任务

本文不讲理论，只讲能直接复制的工作流。

场景一：自动生成周报

每周五最痛苦的事情之一： 翻聊天记录、翻 Notion、翻会议纪要， 然后拼一份“看起来很忙”的周报。

基础 Prompt

你是一位职场写作助手，擅长把零散工作记录整理成周报。

请根据以下内容生成周报：

【粘贴工作记录】

结构：
1. 本周完成事项
2. 进行中的事项
3. 下周计划
4. 需要支持事项

要求：
- 专业简洁
- 突出结果
- 每部分不超过200字

自动化版本

更进一步，你甚至可以让 Codex：

  • 自动读取日历
  • 自动读取 Gmail
  • 自动读取 Notion
  • 每周五自动生成草稿

你只负责最后看一眼。

场景二：Excel 数据处理

大多数职场人不是不会分析数据， 而是不会 Excel 函数。

Codex 最大的价值是： 你不用记公式。

自然语言写公式

我的 Excel 表：

A列：产品名
B列：点击量
C列：成交量

请帮我写一个公式：

计算转化率 = 成交量 / 点击量
显示百分比
保留两位小数
点击量为0时显示“无数据”

Codex 会直接返回： 

=IF(B2=0,"无数据",TEXT(C2/B2,"0.00%"))

更强的是：直接操作 Excel 文件

你甚至可以上传 Excel 文件， 然后直接说： 

把转化率低于5%的行标红，
新增汇总行，
然后返回 result.xlsx

场景三：批量文件处理

“每天重复做的事”，是最适合自动化的。

  • 批量重命名文件
  • 合同信息提取
  • PDF 转 Excel
  • 日报汇总

合同信息提取

读取文件夹中的合同 PDF，
提取以下字段：

- 合同编号
- 甲方名称
- 签约日期
- 合同金额

输出为 Excel 表格。

以前是实习生干的活。 现在是 AI 干。

场景四：职场文档写作

大多数职场文档， 本质问题不是“不会写”， 而是“不知道怎么组织”。

比如：

  • 项目复盘
  • 绩效自评
  • 跨部门邮件
  • 会议纪要

会议纪要 Prompt

请根据以下会议记录，
整理正式会议纪要：

【粘贴原始记录】

输出格式：
1. 会议背景
2. 讨论重点
3. 决策事项
4. 行动项
5. 未决问题

场景五：竞品分析

以前做竞品分析：

  • 开十几个网页
  • 复制粘贴
  • 截图
  • 整理 PPT

现在： 

分析以下竞品：

1. 功能
2. 定价
3. 用户评价
4. 最近动态

输出：
- 对比表格
- 每家总结
- 对我们的启示

Codex 会自己联网搜索、整理、输出。

场景六：Codex 开始操作你的电脑

这是最重要的一次变化。

AI 不只是“帮你想”， 而是开始“帮你做”。

Computer Use

Codex 可以：

  • 看到你的屏幕
  • 操作鼠标键盘
  • 点击按钮
  • 填写表单
  • 打开 ERP/OA
@Computer 打开 ERP 系统，
导出本周销售报表，
整理成 Excel，
然后截图给我确认。

这已经不是聊天机器人了。

这是数字员工。

最后：Prompt 写得好，效率差十倍

OpenAI 官方总结了四件最重要的事：

  • 上下文（背景）
  • 输入素材
  • 输出格式
  • 约束条件

翻译成人话就是：

  1. 你是谁
  2. 你有什么
  3. 你想要什么
  4. 你不要什么
